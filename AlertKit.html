AlertKit

AlertKit.alert

Alert

Basic alert.

AlertKit.alert('Example', 'Hello, world!');

Title

Alert containing only a title.

AlertKit.alert('Hello, world!');

Text

Alert containing only text.

AlertKit.alert(null, 'Hello, world!');

Alert with Buttons

An alert with the default button style.

AlertKit.alert('Example', 'Hello, world!', [['Dismiss', 'AlertKit.alert.close()'], ['Google', 'window.open(`http://google.com/`)']]);

Alert with Custom Buttons

An alert with custom button styles.

AlertKit.alert('Example', 'Hello, world!', [['Dismiss', 'AlertKit.alert.close()', 'FF0000', null, null, 'F0F3F4', null, null, '150ms'], ['Google', 'window.open(`http://google.com/`)', 'FFFFFF', 'F0F3F4', '0000FF', null, '0000FF']])

Buttons

Alert containing only buttons.

AlertKit.alert(null, null, [['Dismiss', 'AlertKit.alert.close()']]);

Title and Buttons

Alert containing only a title and buttons.

AlertKit.alert('Hello, world!', null, [['Dismiss', 'AlertKit.alert.close()']]);

Text and Buttons

Alert containing only text and buttons.

AlertKit.alert(null, 'Hello, world!', [['Dismiss', 'AlertKit.alert.close()']]);

Alert without Click-Out

An alert which doesn't let the user click the background to dismiss it.

AlertKit.alert('Example', 'Hello, world!', [['Dismiss', 'AlertKit.alert.close()']], false);

Alert with HTML

An alert which can contain HTML.

AlertKit.alert('Example', '<i>Hello, world!</i>', [['Dismiss', 'AlertKit.alert.close()']], null, true);

Multiple calls

See how multiple AlertKit.alert calls are handled.

AlertKit.alert('Demo', 'Click the button below to call AlertKit.alert again.', [['Call', 'AlertKit.alert(`Complete`, `Click the background of this alert to dismiss.`)']]);

Timed Alert

An alert automatically dismisses after a configurable period of time.

AlertKit.alert('Example', 'Hello, world!', null, false, false, 2.5);
